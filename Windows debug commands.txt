## ⚙️ Windows Useful Tools & Commands
> Note: This does NOT cover basic tools that are easily accessible in Windows UI.
---

1. 🧰 God Mode
Create a new folder and rename it to: "GodMode.{ED7BA470-8E54-465E-825C-99712043E01C}" 
After renaming, the folder icon will change to the Control Panel icon.  
This gives you quick access to many advanced system settings in one place.

2. Services menu(Don't mess if you don't know about the program)
Run this command "services.msc" or and you can change the startup services.

3. Boot Options (Safe Boot / Troubleshooting) (DON"T MESS UNNECESSARILY OR YOU CAN SAY BYE BYE TO WINDOWS)
Run "msconfig" config to fix your boot and other problems like malware that might run so you can select specific options to only start windows with default safe programs.

4. Edit group policy 
It's used mainly in offices or schools. Controls who have access to administrator controls.

5. Run shell:startup command.(Make your own scripts run after windows run.)
You can find some startup programs here. You can add your own scripts to run after the system starts up.

6. SUPER USER or SUPER ADMIN account(Only use if original admin account not working)
Boot into safe mode and run cmd as administrator.
Type this command "net user administrator /active:yes"
Now, this account is active login to it and then disable after work is done.

Usefull cmd commands 
ipconfig
ping
tracert
dxdiag
winver


### Fix: Stop ISP DNS suffix injection (Windows)
This prevents Windows from appending ISP-provided DNS search domains.

Run in Command Prompt (Admin):
reg add "HKLM\SYSTEM\CurrentControlSet\Services\Tcpip\Parameters" /v SearchList /t REG_SZ /d "."

Revert (if needed):
reg delete "HKLM\SYSTEM\CurrentControlSet\Services\Tcpip\Parameters" /v SearchList /f
